#nginx

Instalación de Nginx en Debian

Guía para instalar y poner en marcha Nginx en sistemas Linux basados en Debian.

Requerimientos

Sistema OperativoVersión
Debian10

Instalación

  1. Actualiza los índices de paquetes.

    sudo apt-get update

  2. Opcional: elimina versiones anteriores.

    sudo apt-get purge nginx nginx-common

  3. Instala la versión más reciente disponible en los repositorios.

    sudo apt install nginx

  4. Inicia el servicio.

    sudo service nginx start

Verificación

Inicio automático

Controla el inicio automático del servicio.

sudo systemctl enable nginx

Para deshabilitarlo:

sudo systemctl disable nginx
