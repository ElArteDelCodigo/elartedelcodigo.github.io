#postgres

PostgreSQL psql Commands

Reference of commonly used psql commands for PostgreSQL 12 database management.

Open the PostgreSQL console before running these examples.

sudo -u postgres psql

psql (12.3 (Debian 12.3-1.pgdg100+1))
Digite «help» para obtener ayuda.

postgres=#
